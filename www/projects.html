Simple HTTP Server

This website uses Simple HTTP Server to server files to clients. Some pages on this website are dynamically generated by the program.

Used Languages

C++, HTML, CSS, git, JavaScript

Used Libraries / Softwares

boost C++ Libraries, JQuery, MySql, CMake, Ubuntu

More Details

This is making on Ubuntu and working on Ubuntu server on Google Cloud. For network programming, I use asio from boost C++ Libraries, and for now, GET and POST are implemented (not fully). I also use filesystem from boost C++ Libraries for making folder and making dynamic page(Files Page). On Request Log Page, I use MySql for my databese. This is build with CMake and Request Log Page use JQuery for radio buttons and changing body width.

Find and Move

A PhD student had some problem and asked me to write a program for it. She had 2 big csv files and wanted to move some columns of data from one file to other when one of columns in one file matches to the one in other file. However, the colums in rows in the big files do not exactly match (they could) because it was modified little bit.

This program finds rows that almost / exactly match, and moves some colums of data from one file to the other.

Used Languages

C++

Used Libraries / Softwares

Visual Studio, Windows

More Details

This is made using Visual Studio. Threads are used to make it faster to complete the task. To find rows that almost match, I use edit distancealgorithm. Because 2 rows do not exactly match (they could) so normal string comparison did not work.

Source Code on GitHub

https://github.com/kokay/SimpleHttpServer

Demo

On the demo, the program compares a column 'comment' in both files for all rows. If the colomns almost / exactly match, then the program copies columns after a column 'info1' in the row in 'moveFrom.csv' to the row in 'moveTo.csv'. Actually to keep the original files the result is written in result.csv, and which rows matches are written in log.csv.
